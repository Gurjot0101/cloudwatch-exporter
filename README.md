# CloudWatch Exporter Script

This repository contains a script to export AWS CloudWatch logs to an S3 bucket for archival, analysis, or downstream processing.

## 🚀 Features
- Fetches CloudWatch log groups and streams
- Supports scheduled or manual runs
- Simple configuration via environment variables
